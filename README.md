## ğŸ“ Programs Overview

### P1.html - Lists, Tables & Clickable Images

Demonstrates the implementation of:

- **Unordered Lists** (`<ul>`)
- **Ordered Lists** (`<ol>`)
- **Description Lists** (`<dl>`)
- **HTML Tables** with `rowspan` and `colspan` attributes
- **Clickable Images** using anchor tags

### P2.html - Forms & Frames

A complete form implementation featuring:

- Text inputs (first name, last name, address)
- Radio buttons for gender selection
- Email input with validation
- Date picker for date of birth
- Form submission with JavaScript handling
- Dynamic output display

### P3.html - CSS Styling

Demonstrates various CSS styling techniques:

- Inline styles
- Internal stylesheet (`<style>` tag)
- External stylesheet linking (`style.css`)
- Font properties (family, size, weight, style)

### P4.html - Window Object Functions

Interactive demonstration of JavaScript Window object methods:

- `window.alert()` - Display alert dialog
- `window.prompt()` - Get user input
- `window.confirm()` - Confirmation dialog
- `window.open()` - Open new browser window
- `window.setTimeout()` - Delayed execution

### P5.html - DOM Manipulation

Demonstrates dynamic DOM manipulation:

- Changing element styles via JavaScript
- Event handling with `onclick`
- `getElementById()` method
- Dynamic background color changes
